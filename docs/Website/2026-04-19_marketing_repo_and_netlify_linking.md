# Marketing Repo + Netlify Linking

Date: 2026-04-19

## Scope
This repository is the dedicated source for the FlowIQ marketing website.

## Canonical Targets
- GitHub repo: `https://github.com/Quintencd/freightiq-website`
- Netlify site: `flowiq-website`
- Site URL: `https://www.flowiq.info`
- Site ID: `e6a1f747-13f1-4856-83c1-b69104047578`

## Workflow
1. Make website edits in this repo only.
2. Commit and push to `main` in `freightiq-website`.
3. Deploy from this repo folder with:
   - `npm run deploy:preview`
   - `npm run deploy:prod`

## Guardrail
Do not deploy marketing changes from the app repo (`Quintencd/Freightiq`).
